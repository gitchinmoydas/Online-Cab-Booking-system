**Zyber-Cab** is a full-stack, real-time ride-hailing application inspired by platforms like Uber and Ola.  
Built with the **MERN** stack (MongoDB, Express.js, React, Node.js), Zyber-Cab allows users to **book rides**, **track captains live**, and **schedule trips** — all from a modern, intuitive interface.

---

## 🚀 Features

- 🔐 **Secure Authentication** (JWT, bcrypt)
- 📍 **Real-Time Captain Tracking** (Socket.IO)
- 🚖 **Instant Ride Booking & Broadcasting** to nearby Captains
- 🧭 **Basic Route Optimization**
- 📅 **Ride Scheduling & Pre-Booking**
- 📦 **Clean MVC Backend Architecture**
- 🖥️ **Modern React Frontend (Tailwind + Shadcn UI)**
- 💳 **Payment Integration (Coming Soon)**
